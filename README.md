# Ken Chu Portfolio

Personal portfolio website for AI engineering, production automation systems, computer vision, edge AI, and robotics-adjacent projects.

Live site: https://kenchudigital.github.io

## Tech Stack

HTML · CSS · JavaScript · GitHub Pages

## Local Preview

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Source

This website is based on the open-source vCard personal portfolio template by CodeWithSadee:

https://github.com/codewithsadee/vcard-personal-portfolio

The content, projects, and profile have been customized for my personal portfolio.
